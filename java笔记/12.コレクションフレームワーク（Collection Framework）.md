# 一、コレクションの種類
## 1）ArrayList
下記のコードのように：
```java
ArrayList<String> list = new ArrayList<>();  
list.add("java1");  
list.add("java2");  
list.add("java1");  
list.add("java2");  
System.out.println(list);
```
重複できるし、索引も支持しいている
## 2）HashSet
代码如下：
```java
//重複のエレメントを支持しないし、索引もない  
HashSet<String> set = new HashSet<>();  
set.add("java1");  
set.add("java2");  
set.add("java1");  
set.add("java3");  
System.out.println(set);
```
